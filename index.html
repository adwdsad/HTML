Абсолютно все типы логики JavaScript

Как использовать свои цифры: Чтобы считать введенное число в коде, используйте встроенную функцию getNum(). Она автоматически возьмет цифру из поля над редактором.

1. Переменные и Арифметика

Зачем нужно: Хранение данных в памяти и выполнение расчетов. Измените цифры в полях ниже, чтобы сложить их вашим кодом.

Шаблон для проверки (скопируйте в поле ввода):

let a = getNum("v1"); // Считает первое число
let b = getNum("v2"); // Считает второе число
log("Результат сложения: " + (a + b));

2. Условные операторы (If / Else)

Зачем нужно: Разветвление программы. Проверим введенное вами число на соответствие условию (например, возраст или баланс счета).

Шаблон для проверки:

let n = getNum("c1");
if (n >= 18) { log("Число больше или равно 18"); } else { log("Число меньше 18"); }

3. Циклы (Loops)

Зачем нужно: Повторение действий. Задайте число в поле ввода, чтобы определить, сколько именно кругов (итераций) должен сделать цикл.

Шаблон для проверки:

let limit = getNum("l1");
for (let i = 1; i <= limit; i++) {
log("Повторение №" + i);
}

4. Массивы (Списки)

Зачем нужно: Группировка данных. Вы можете указать числовой индекс элемента, который хотите извлечь из созданного списка.

Шаблон для проверки:

let index = getNum("a1");
let shop = ["Хлеб", "Молоко", "Сыр", "Масло"];
log("Выбранный товар: " + shop[index]);

5. Объекты

Зачем нужно: Структурирование информации. Задайте новую цену в поле, чтобы динамически обновить стоимость внутри объекта.

Шаблон для проверки:

let newPrice = getNum("o1");
const item = { name: "Ноутбук", price: 50000 };
item.price = newPrice; // Перезапись свойства объекта
log(item.name + " теперь стоит " + item.price + " руб.");

6. Асинхронность и Таймеры

Зачем нужно: Управление временем. Задайте количество миллисекунд (1 сек = 1000 мс), чтобы настроить таймер через поле ввода.

Шаблон для проверки:

let delay = getNum("as1");
log("Запуск...");
setTimeout(() => { log("Таймер сработал!"); }, delay);

7. Перехват ошибок (Try / Catch)

Зачем нужно: Предотвращение падений кода. Если ввести в поле 0, наш код умышленно вызовет ошибку деления, но сайт продолжит работать.

Шаблон для проверки:

let divider = getNum("e1");
try {
if(divider === 0) { throw new Error("Деление на ноль!"); }
log("Результат: " + (100 / divider));
} catch(err) { log("Поймана ошибка: " + err.message); }
